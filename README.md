# Simple menu app

## Installing
1. `python3.6 -m venv .venv`
2. `source .venv/bin/activate`
3. `pip install -r requirements/local.txt`
4. `./manage.py migrate`
5. `./manage.py createsuperuser` (in order to access admin page)
6. `./manage.py runserver`

## Running the tests
`pytest`
